# `NYC-dashboard` — A simple dashboard for the NYC citi bike data

## Getting Started

To get you started you can simply clone the `NYC-dashboard` repository and install the dependencies:

### Clone `NYC-dashboard`

Clone the `NYC-dashboard` repository using git:

```
git clone https://github.com/Umer0114/NYC-dashboard
cd NYC-dashboard
```

### Install Dependencies

* Npm and bower are used as package managers, so run the following commands to get the dependencies.

```
npm install
bower install
```


### Run the Application

* Run the following command to get the project running.

```
npm start
```

Now browse to the app at [`localhost:8000/`].

### Test the application

* The application accompanies the necessary unit tests. To execute the unit-tests, run the following command.

```
npm test
```
